¿Qué es "El Garage de Lis"? 

"El Garage de Lis" es una tienda virtual de vehículos diseñada para ofrecerte una experiencia de compra rápida, segura y moderna. Aquí podrás explorar una amplia variedad de autos nuevos y usados, filtrarlos según tus preferencias, agregarlos a tu carrito y simular una compra con factura digital.

¿Qué puedes hacer en esta página?

1. Explorar Vehículos
Visualiza una galería de autos con imágenes, marcas, modelos, categorías y precios.
Cada vehículo incluye detalles como tipo (sedán, SUV, camioneta, etc.) y categoría (nuevo, usado, premium).

2. Buscar y Filtrar
Usa la barra de búsqueda para encontrar vehículos por marca, modelo o categoría.
Los resultados se actualizan al instante mientras escribes.

3. Ver Detalles
Haz clic en "Ver Detalle" para abrir una ventana emergente con una imagen ampliada y toda la información del vehículo.

4. Añadir al Carrito
Selecciona la cantidad de vehículos que deseas comprar y agrégalos a tu carrito con un solo clic.
El carrito muestra un resumen de tus compras, incluyendo imágenes, cantidades y subtotales.

5. Simular una Compra
Completa un formulario de pago con tu nombre y número de tarjeta (simulado).
Al confirmar, se genera una factura en PDF con los detalles de tu compra, que podrás descargar y guardar.

Tecnologías Utilizadas:

Esta página web está construida con las últimas tecnologías de desarrollo frontend:

HTML5: Para una estructura semántica y accesible.

CSS3 y Bootstrap 5: Para un diseño responsivo, moderno y adaptable a cualquier dispositivo.

JavaScript (ES6+): Para una experiencia dinámica e interactiva.

Font Awesome: Para iconos visualmente atractivos.

jsPDF: Para generar facturas en formato PDF.
